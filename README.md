# TigerTix
## Sprint 1
### Backend
Clean, setup, and run the backend by navigating to the "backend" folder and executing the following command:
```sh
npm run all
```

(or, just run it with `npm run start` when the database has already been created)

### Frontend
Run the frontend by navigating to the "frontend" folder and executing the following command:
```sh
npm run start
```
