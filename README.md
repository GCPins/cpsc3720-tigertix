# TigerTix
### Backend
Clean, setup, and run the backend by navigating to the "backend" folder and executing the following commands:
```sh
# install the node modules
npm install

# setup database
npm run setup

# run the admin and client backends
npm run all
```

Or, just run it with the `npm run start` command after the node modules have been installed and the database has already been created.

### Frontend
Run the frontend by navigating to the "frontend" folder and executing the following command:
```sh
# install the node modules
npm install

# run the frontend
npm run start
```
